World Optimizer
===============

概要
----
World Optimizerは、VRChatやUnityプロジェクト向けの強力な最適化ツールです。
複雑な設定を必要とせず、ワンクリックでワールドのパフォーマンスを最大限に引き出します。

主な機能
--------
- ワンクリック最適化
  メッシュ結合、テクスチャ圧縮、不要なオブジェクトの削除などを一括で実行します。

- 詳細な分析 (Scene Analyzer)
  シーン内のボトルネックを自動で検出し、改善点をリストアップします。

- 安全なバックアップ
  変更を加える前に資産のバックアップを自動生成するため、安心して最適化を行えます。

- 個別実行モード
  「テクスチャのみ」「オーディオのみ」など、必要な処理だけを選んで実行することも可能です。

インストール方法
----------------
Unityプロジェクトの `Assets` フォルダに `WorldOptimizer` フォルダを配置してください。
メニューバーに [Tools] -> [World Optimizer] が表示されます。

動作環境
--------
- Unity 2019.4 LTS 以降
- VRChat SDK 対応 (World SDK 3.0推奨)
